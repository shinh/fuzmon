## Style

- Remove all compiler warnings
- Use info! log for non-frequent events and use warn! for errors
- Never copy-and-paste code. Eliminate duplicated code
- Run `cargo fmt`
- Never use `allow(dead_code)`

## Testing

- Add appropriate tests
- Get `cargo test` pass
